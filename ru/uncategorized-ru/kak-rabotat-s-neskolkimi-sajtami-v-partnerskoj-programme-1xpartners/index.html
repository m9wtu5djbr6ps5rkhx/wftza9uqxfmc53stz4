Как работать с несколькими сайтами в партнерской программе 1xPartners

Published: 04.03.2024

CATEGORY MENU

  • Uncategorized 15

Если вы являетесь вебмастером, у которого более 1 ресурса, сотрудничающего с 1xPartners, то, конечно, вас интересует детализация статистики. Отслеживать переходы, регистрации и депозиты важно для понимания своей аудитории и источников прибыли вне зависимости от того, какие партнерские программы букмекерских контор вы выбираете. Ниже детально описано, как работает партнерка 1xBet в плане статистики и как получить нужные данные по вашим сайтам.

Как добавить несколько ресурсов


Чтобы разделить вашу аудиторию по источникам трафика, будь то сайты, соцсети или другие площадки, используйте партнерскую программу 1xPartners. Для этого в главном меню выберите пункт «Добавить сайт». После чего добавьте все ресурсы, которые у вас есть. При этом вам не обязательно добавлять домены сайтов, название ресурса может быть произвольным, скажем, «Группа футбол VK». После добавления всех площадок, с которых вы планируете вести трафик, партнерка от 1xBet дает возможность сформировать свою ссылку для каждого ресурса.

Для этого необходимо выполнить следующие шаги:

1. Войдите в раздел «Партнерские ссылки»

2. В выпадающем списке «Сайт» выберите созданную ранее площадку

3. Создайте партнерскую ссылку

Этот алгоритм надо выполнить с каждой площадкой.

Как посмотреть статистику по вашим ресурсам


По каждому сайту партнерка с 1xbet также позволяет просмотреть статистику по регистрациям, переходам, кликам и депозитам. Для этого перейдите в раздел «Полный отчет». В поле построения отчета выберите следующие параметры:

Валюта – та валюта, в которой у вас созданы партнерские ссылки.

Сайт – площадка, статистика по которой вас интересует.

Дата или диапазон дат – временной отрезок, данные по которому вам нужны.

Подобный отчет даст возможность посмотреть подробную статистику по конкретной площадке. Если же вы хотите изучить данные по всем площадкам, которые указали, то партнерка 1xBet дает и такую возможность. Для этого выставьте в поле «Сайт» параметр «Все сайты». Если вы добавили все площадки и создавали ссылки под каждый из ресурсов, то увидите детальную статистику в одной сводной таблице.

Как разделить статистику в рамках одного домена


Чтобы понимать, какая страница, раздел, пост или видео ведет к лучшей конверсии, надо смотреть данные непосредственно по ресурсам одной площадки. Что для этого делать? Используйте параметр «SubID». Когда создаете ссылку, то кроме выбора сайта введите в поле «SubID» метку для ссылки.

Как отслеживать статистику по SubID


После разделения трафика по SubID посмотреть отчет по конкретной аудитории можно 2 способами:

1. Как отчет по игрокам

Здесь вы можете увидеть, с какой страницы или из какого раздела пришли те или иные игроки.

2. Как отчет по промоматериалам

Для этого вы можете отсортировать общую статистику по SubID.

Альтернативные способы просмотра статистики


В качестве альтернативы партнерской программе вы можете использовать CPA-трекеры. В этом случае вы можете создать одну партнерскую ссылку и настроить сбор данных в соответствии с вашими пожеланиями по детализации статистики.
